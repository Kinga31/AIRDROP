### Quick Start

The site is deployed on Vercel at [airdropalpha.vercel.app](https://airdropalpha.vercel.app)

To view analytics data:
1. Visit the [Vercel Dashboard](https://vercel.com/dashboard)
2. Select the **airdropalpha** project
3. Click the **Analytics** tab
